## About this dataset

The **Rwamagana Imihigo 2024–2025** dataset captures the district's annual performance contract commitments and quarterly progress. It was created on **2025-06-03** and last updated on **2025-08-05**.

For each performance indicator the dataset contains:

- A **Target** element — the value committed to at the start of the fiscal year
- An **Achieved** element — the actual result reported each quarter
